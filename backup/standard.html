Plumbious主题的标准：

  • 必须且目前只允许包含：
    • default.html
    • index.html
    • post.html
    • archives.html
    • categories.html
    • tags.html
    • 404.html
  • meta需要包含以下内容： 
    <meta name="author" content="{{ site.author }}"/>
<meta name="keywords" content="{% if page.tags %}{{ page.tags }}{% else %}{{ site.keywords }}{% endif %}" />
<meta name="description" content="{% if page.description %}{{ page.description }}{% else %}{{ site.description }}{% endif %}" />
<meta name="viewport" content="width=device-width; initial-scale=1.0">（可修改initial-scale）
  • 目前支持三种主页显示：
    • 1~3篇摘要+其他日志列表
    • 1~3篇日志简介+其他日志列表
    • 纯日志列表
  • 允许在default.html中的页脚添加主题来源等信息
  • 支持多种界面浏览，优秀的手机浏览效果

Markdown顶部书写格式：

/* 以下内容全部顶格写，之前一定不允许有空格，冒号后一定要有空格 */
---
layout: post //不可更改
title: //文章标题
description: //文章简介，为了让搜索引擎检索到你的文章的简介，简介在60字以内即可
categories: //文章分类
tags: //文章标签，也会被是为文章的关键字，为了让搜索引擎检索到你的文章的关键字
---
