如何打开终端？

Windows

使用快捷键ctrl+r打开运行，然后敲入CMD，回车即可打开终端。

Mac

English：Aplication > Utilites > Terminal

中文：应用程序 > 实用程序 > 终端

或可以搜索终端/Terminal

Ubuntu

中文：应用程序 > 附件 > 终端

或可以搜索终端/Terminal

如何使用Prose撰写博文？
  1. 进入你的Repositories，打开_post文件夹
  2. 点击右上角的+ NEW FILE
  3. 编辑_posts/XXXX-XX-XX-your-filename.md中的“your-filename”为日志的连接，一般为英文，请用-代替空格
  4. 点击进入Metadata，粘贴一下元素：（冒号后面一定要有空格）
    layout: post
title: 文章标题
description: 文章简介，为了让搜索引擎检索到你的文章的简介，简介在60字以内即可
categories: 文章分类
tags: 文章标签，也会被是为文章的关键字，为了让搜索引擎检索到你的文章的关键字
  5. 点击铅笔按钮，开始使用Markdown语言写博文
  6. 点击眼睛按钮可以预览文章
